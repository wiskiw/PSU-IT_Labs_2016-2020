Тэги <td>...</td> и <th>...</th>

Служат для создания объекта ячейка

Может иметь атрибуты:
 
Атрибут Возможные значения Описание Значение по умолчанию
align left, right, center, justify Выравнивание текста внутри ячеек.
По левому краю, по правому краю, по центру, по обоим краям одновременно (по ширине) 		по левому краю
valign top, bottom, baseline, middle Вертикальное выравнивание текста в ячейке.
По верхнему краю, по нижнему краю, по центру (по центральной линии ячейки), распределить по высоте		 baseline
width X, Xpt, Xpx, X%,
где X - число		 Ширина ячейки в пунктах, пикселях или в процентах от ширины таблицы. Определяет ширину столбца, причем для задании ширины столбца достаточно указать ширину только одной из ячеек в этом столбце.

ВАЖНО!
Указывать ширину заголовочной ячейки <th>...</th> нежелательно, т.к. при этом выравнивание текста в ней может отображаться некорректно.

Ширина может быть указана как для одной ячейки в столбце, так и для каждой внутри этого столбца. Разумеется, при этом появляется довольно большое количество избыточной информации.
В случае, когда внутри столбца для ячеек указываются разные значения ширины, столбец принимает наибольшую ширину из указанных, а все остальные значения ширин для других ячеек в столбце - не просто избыточная, но и бесполезная информация.

 нет
height X, Xpt, Xpx, X%,
где X - число		 Высота ячейки в пунктах, пикселях или в процентах от высоты таблицы. Если единицы не указываются, то браузер считает, что высота указана в пикселях. нет
bgcolor цвет Цвет фона ячейки нет
bordercolor цвет Цвет границы вокруг ячейки черный
title текст Текст всплывающей подсказки нет
collspan= число Количество объединяемых ячеек по горизонтали (по столбцам)
При указании этого параметра количество ячеек в строке (количество контейнеров <td>...</td> или <th>...</th>) сокращается на указанное число объединяемых ячеек		  
rowspan= число Количество объединяемых ячеек по вертикали (построчно).
При указании этого параметра количество ячеек в объединяемых строках сокращается на единицу		 нет

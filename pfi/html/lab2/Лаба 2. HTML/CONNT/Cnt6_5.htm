Задание 2

Из тех HTML-страниц, что были созданы Вами на всем протяжении
изучения языка HTML, создайте сайт
на основе фреймовых структур с использованием обыкновенных фреймов.

Для этого:

  1. в Вашей папке создайте файл index_frame.html.
  2. в нем создайте фреймовую структуру.
    Напоминаю, что веб-страницы с фреймовыми структурами не должны содержать контейнер <BODY>...</BODY>!
    Структура должна состоять из двух вертикальных фреймов.
  3. левый фрейм шириной 150-250 пунктов (на Ваш выбор) назовите menu, правый - frame_cont
  4. создайте еще один файл menu.html, в котором разместите список ссылок из предыдущего задания\
  5. в этом списке измените атрибуты target=content на target=frame_cont
  6. создайте еще одну - стартовую - страницу, и назовите ее start.html. Напишите в ней текст приветственного сообщения для посетителей Вашего сайта
  7. в файле index_frame.html в левый фрейм в качестве значения атрибута src задайте адрес страницы menu.html, а в правый - адрес стартовой страницы start.html.

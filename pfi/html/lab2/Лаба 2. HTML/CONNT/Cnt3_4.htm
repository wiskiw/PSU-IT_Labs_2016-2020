Всплывающая подсказка

Для повышения информативности картинки ее можно снабдить всплывающей подсказкой - краткой текстовой надписью, которая появляется при наведении указателя мыши на картинку после некоторой задержки.

Текст всплывающей подсказки можно задать в атрибуте title:

<IMG title="Текст всплывающей подсказки" src="имя_файла">

Примечание: атрибут title допустимо применять не только для тега <IMG>, но и для других объектов веб-страницы. Например, пункты меню данного документа

Задание 1

Из тех HTML-страниц, что были созданы Вами на всем протяжении
изучения языка HTML, создайте сайт
на основе фреймовых структур с использованием плавающего фрейма.

Для этого:

  1. в Вашей папке создайте файл index_iframe.html.
  2. в нем создайте таблицу из 2х столбцов и 1-й строки. Для таблицы задайте высоту и ширину 100%, а также толщину границ между ячейками в 1 пункт (временно). Для первой ячейки задайте ширину 150-250 пунктов (на Ваш выбор)
  3. в левой ячейке разместите список Ваших web-страниц в виде нумерованного списка гиперссылок на Ваши задания. Текст гиперссылок должен соответствовать названиям заданий на русском языке (можно воспользоваться уже готовым аналогичным списком из предыдущего задания).
  4. в правой ячейке разместите контейнер <iframe>...</iframe>. Задайте для этого плавающего фрейма имя content. Настройте значения высоты и ширины для этого фрейма (100%)
  5. для каждой ссылки в левой ячейке укажите дополнительный атрибут target и направьте его в созданный фрейм

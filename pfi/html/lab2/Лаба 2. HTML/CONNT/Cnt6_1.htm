Фреймовые структуры

Фреймовые структуры - структуры, при которых рабочая область окна браузера разбивается на отдельные области каждая из которых может являться самостоятельным окном. Фрейм - это и есть та самая область окна, которая как раз и может являться самостоятельным окном. Иными словами фрейм - это окно в окне.
Фреймов на странице может быть несколько

Если Вы посмотрите в адресную строку браузера, то увидите, что при переходе по ссылкам адрес остается неизменным. И действительно, при работе с фреймами переходы по ссылкам осуществляются не в главном окне браузера, а внутри фрейма.

Фреймы бывают плавающими и обыкновенными. Обыкновенные фреймы используют всю рабочую область окна полностью, деля ее по горизонтали или по вертикали, при этом одна фреймовая структура может быть вложена в другую. Плавающие фреймы - прямоугольные контейнеры, организованные с помощью тегов <iframe> ... </iframe>, в которые может быть загружена та или иная страница. С плавающими фреймами работать проще, чем с обыкновенными

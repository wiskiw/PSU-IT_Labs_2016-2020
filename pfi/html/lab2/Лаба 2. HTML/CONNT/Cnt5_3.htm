Задание 1

Создайте HTML-документ по приведенному образцу




При выполнении этого задания необходимо:

  1. Разметить текстовые блоки абзацами
  2. Выделить заголовки абзацев тегами <H2>...</H2>
  3. Пометить заголовки абзацев закладками (тегами <a name="{Вами_придуманное_имя_закладки}">...</a>)  
  4. Выписать заголовки абзацев в столбец в начало текста, представить его в виде маркированного списка, каждый пункт которого выражен гиперссылкой.
  5. Нацелить гиперссылки на соответствующие закладки в тексте

Обыкновенные фреймы

Обыкновенные фреймы создаются в отдельных страницах, которые являются каркасом для всего сайта.
Причем - что ОЧЕНЬ ВАЖНО (!) - на таких страницах контейнер <BODY>...</BODY> не создается! Вместо него используется контейнер <frameset> ...</frameset>

 

Теги для оформления обыкновенных фреймов

Тег Описание
<frameset> ...</frameset> Главный контейнер. Размечает окно браузера на фреймы
<frameset rows="x, y, ..."> ...</frameset> Разбивка окна по горизонтали. Числа x, y, ... - задание высоты фрейма. Количество горизонтальных фреймов может быть разным.
Рекомендуется для одного из создаваемых фреймов высоту указывать не числом, а символом «*». В этом случае фрейм займет все пространство окна, оставшееся после выделения фреймов с явно заданными высотами
<frameset cols="x, y, ..."> ...</frameset> Аналогично rows, но только здесь разбивка окна производится по вертикали.
<frameset frameborder="yes | no"> ...</frameset> Наличие рамки между фреймами. Слова yes и no можно заменить цифрами 1 и 0 соответственно
<frameset border=N> ...</frameset> Толщина рамки между фреймами. N - число. При N=0 рамки не видно, но она есть.
При использовании вместе с атрибутом frameborder="no" теряет всякий смысл
<frameset framecolor="цвет"> ...</frameset> Задание цвета рамки
<frameset ...>
<frame>
</frameset>		 Создание и настройка параметров каждого из фреймов.
Количество тегов <frame> должно соответствовать их количеству в родительском контейнере <frameset> ...</frameset>
<frame name="имя"> Задание имени фрейма. По этому имени можно направлять открываемые по гиперссылкам страницы в нужные фреймы с помощью атрибута гиперссылки target
<frame src="имя_файла"> Изначально загружаемая в данный фрейм веб-страница
<frame noresize> Запрет на изменение размеров фрейма
<frame scrolling="yes | no | auto"> yes - полосы прокрутки видны всегда, no - полосы прокрутки отсутствуют, auto - полосы прокрутки появляются при необходимости
<frame marginwidth=N> Поля по горизонтали. По умолчанию (если не указано) 10 пикселей
<frame marginheight=N> Поля по вертикали. По умолчанию (если не указано) 10 пикселей

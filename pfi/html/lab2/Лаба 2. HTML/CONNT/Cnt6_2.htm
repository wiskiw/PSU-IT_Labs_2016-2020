Пример использования плавающего фрейма - данный практикум.

Атрибут Возможные значения Описание Значение по умолчанию
src URL или локальное имя файла подгружаемой веб-страницы Этот атрибут не является обязательным и предназначен лишь для первоначальной загрузки веб-страницы во фрейм. Любая ссылка с указанным атрибутом target может направить в этот фрейм страницу, на которую ссылается нет
name Имя фрейма При указании имени фрейма появляется возможность загружать в него другие страницы. Имя фрейма должно состоять из латинских символов и не содержать пробелов нет
width X, Xpt, Xpx, X%,
где X - число		 Ширина фрейма в пунктах, пикселях или в процентах от ширины окна. Если единицы не указываются, то браузер считает, что ширина указана в пикселях. нет
height X, Xpt, Xpx, X%,
где X - число		 Высота фрейма в пунктах, пикселях или в процентах от высоты окна. Если единицы не указываются, то браузер считает, что высота указана в пикселях. нет
hspace X, Xpt, Xpx, X%,
где X - число		 Поля по горизонтали нет
vspace X, Xpt, Xpx, X%,
где X - число		 Поля по вертикали нет
align left, right, center Выравнивание фрейма по отношению к соседним элементам на веб-странице.
По левому краю, по правому краю, по центру		 нет

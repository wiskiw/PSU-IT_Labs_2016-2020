Выравнивание картинки

Для выравнивания картинки используются атрибут align, который может принимать значения:

<IMG align=top | align=middle | align=bottom | align=left | align=right src="имя_файла">

Этот параметр определяет положение картинки по отношению к соседним элементам:

  • top - вертикальное выравнивание по верхнему краю
  • middle - вертикальное выравнивание по центру
  • bottom - вертикальное выравнивание по нижнему краю
  • left - горизонтальное выравнивание по левому краю
  • right - горизонтальное выравнивание по правому краю
  • center - горизонтальное выравнивание по центру

Примечаниение: при всем многообразии значений атрибута align позиционирование картинки на странице лучше осуществлять, внедряя ее в ячейки таблицы, либо в блок <div>...</div>

Тэг <a>...</a>

Сам по себе (без атрибутов) этот контейнер никак на отображение текста внутри него не влияет, смысла как такового не имеет и не используется.
Основными атрибутами тега <a> ... </a> являются:

Атрибут Возможные значения Описание Значение по умолчанию
href URL, имя файла, #имя закладки, e-mail или просто символ # Адрес, по которому происходит отсылка пользователя при клике на гиперссылке. Атрибут необязательный, но используется практически всегда, т.к. только благодаря ему указатель мыши изменяется при его наведении на гиперссылку нет
name #Имя закладки Единственный случай, когда атрибут href можно не использовать. Это вовсе не означает, что атрибуты href  и name несовместимы. Просто таким образом обычно создаются закладки - специальные именованные метки, на которые потом можно направить гиперссылку.
Сама по себе закладка визуально ничем не отличается от окружающего ее текста и вообще никак себя не проявляет		 нет
target _blank, _parent, _search,
_self, _top, #имя_закладки
или имя_фрейма		 Для простых локальных, не отягощенных фреймовыми структурами, апплетами или скриптами страниц смысл значений _parent, _search, _self, и _top сводится к одному (последнему) и обозначает  прокрутку страницы в самый ее верх.
_blank означает, что ссылка должна отрыться в новом окне.
#имя_закладки означает переход к закладке.
Если страница содержит фреймовые структуры, то атрибут имя_фрейма открывает ссылку во фрейме с соответствующим именем (фрейм - внедренное в окно основного документа окно другого документа).
Если страница открыта с помощью атрибута target =_blank, то ссылка с атрибутом target =_parent в новом окне откроет страницу в том окне, из которого она была открыта.
Аналогичный смысл имеют и другие значения атрибута target, о которых речь пойдет позже		 нет
title Текст подсказки Всплывающая подсказка нет
 

Атрибутов у тега <a>...</a> еще довольно много. Некоторые из них являются общими, т.е. могут применяться не только с этим, но и с другими тегами. Некоторые - применяются только при создании гиперссылок. В целом для нормальной работы с гиперссылками достаточно уметь правильно использовать три вышеуказанных атрибута.

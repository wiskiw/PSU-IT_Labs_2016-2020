Таблицы на веб-страницах

Таблицы в HTML представляют собой довольно сложный объект,
в том отношении, что он состоит из трех вложенных друг в друга контейнеров

  • Контейнер таблица - тэг <table>...</table> - корневой контейнер, в который вкладываются дочерние контейнеры строк
  • Контейнер строка - тэг <tr>...</tr> - дочерний контейнер по отношению к тэгу <table>...</table> и родительский по отношению к контейнеру ячейка
  • Контейнер ячейка - тэг <td>...</td> - дочерний контейнер по отношению к тэгу <tr>...</tr>. Организует объект ячейка
    Для организации заголовочных ячеек (с предустановленными параметрами форматирования текста внутри ячейки - выравнивание по центру, полужирное начертание) может использоваться тег <th>...</th>
  • Контейнер заловок таблицы - тэг <caption>...</caption> - необязательный дочерний контейнер по отношению к тэгу <table>...</table>, предназначенный для вывода заголовка таблицы. Обычно располагается перед объявлением первой строки

Важно!
Использование указанных тэгов как-то иначе ни к какому результату не приведет! В лучшем случае появится обычный текст

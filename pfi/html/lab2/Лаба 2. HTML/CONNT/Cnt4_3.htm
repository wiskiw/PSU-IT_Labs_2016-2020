Служит для создания объекта строка

Может иметь атрибуты:
 
Атрибут Возможные значения Описание Значение по умолчанию
align left, right, center, justify Выравнивание текста внутри ячеек.
По левому краю, по правому краю, по центру, по обоим краям одновременно (по ширине) 		по левому краю
valign top, bottom, baseline, middle Вертикальное выравнивание текста в ячейке.
По верхнему краю, по нижнему краю, по центру (по центральной линии ячейки), распределить по высоте		 baseline
width X, Xpt, Xpx, X%,
где X - число		 Ширина строки в пунктах, пикселях или в процентах от ширины таблицы.
Обычно не указывается, т.к. ширина строки задается шириной таблицы.		 нет
height X, Xpt, Xpx, X%,
где X - число		 Высота строки в пунктах, пикселях или в процентах от высоты таблицы. Если единицы не указываются, то браузер считает, что высота указана в пикселях. нет
bgcolor цвет Цвет фона ячеек в этой строке нет
bordercolor цвет Цвет границы вокруг ячеек в этой строке черный
title текст Текст всплывающей подсказки нет

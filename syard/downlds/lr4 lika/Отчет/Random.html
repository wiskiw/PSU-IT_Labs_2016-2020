ГСЧ в диапазоне -n ... 10n типа DBL

Для выполнения задания создан виртуальный подприбор (ВПП), генерирующий случайные числа типа DBL в диапазоне, определяемым формулой -1...10n. Предельные значение Nmin и Nmax от значения числа n, вводимого в элемент управления "Число n".

При правильном выполнении данного ВПП, после ввода числа n и после запуска ВПП щелчком по кнопке Run на индикаторах Nmin и Nmax должны отобразиться предельные значения заданного диапазона случайных чисел, а на индикаторе "Результат" должно появиться случайное число из заданного диапазона.

---------------------- Задание 1 ------------------------------------
				Условие
	Приведите пример нетривиальных выражений, принадлежащих
следующему типу:
	[[[(Integer,Bool)]]]
Требование нетривиальности в данном случае означает, что
встречающиеся в выражениях списки должны содержать больше
одного элемента.

				Решение
1) snd (head (tail (head (head [[[(42, True), (3, False)], [(333, True)]]]))))
// Вывод второго значения котежа из самого первого вложенного массива кортежей.

2) (123, False): (tail (head (head [[[(42, True), (3, False)], [(333, True)]]])))
// Добавление эламента (123, False) во второй вложенный масив кортежей



		
---------------------- Задание 2 ------------------------------------
				Условие
	Определите следующие функции:
	Функция bothTrue :: Bool -> Bool -> Bool, которая возвращает
True тогда и только тогда, когда оба ее аргумента будут равны
True. Не используйте при определении функции стандартные
логический операции (&&, || и т.п.).

				Решение
bothTrue :: Bool -> Bool -> Bool
bothTrue a b = if (a) then (if(b) then True else False) else False 
	// если значение а - True, проверяем значени b; иначе возвращаем False


				Тестирование
	Ввод					Вывод
 bothTrue True True		True
 bothTrue True False		False
 bothTrue False False		False
 bothTrue False True 		False
